Using assets

Including assets such as images is possible by using the following technique.

  1. Create an assets folder directly under the src folder.
  2. In any of your HTML, CSS, and SCSS files, reference any file in the src/assets/ folder using ~/assets/ before your filename. For example, if you added src/assets/photo.jpg, you could reference the image in an HTML file using <img src="~/assets/photo.jpg" alt="Photo" />.

Filenames are updated at build time to include a hash of the file's contents. This means you must use the steps described above to reference any assets.

During the CI build, any assets are automatically uploaded to our CDN and the `--assets` is set to the CDN.
